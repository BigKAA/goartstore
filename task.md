# Artstore redisign

Тестовый проект в old_artsore слишком сложный. Необходимо провести переработку проекта.

- Разделить модули на отдельные независимые проекты по одному на директорию
- Провести ревизию модулей по отдельности с точки зрения API, конфигурации.
  - Подготовить рекомендации по унификации API
  - Зафиксировать контракт по API
  - На основании контракта подготовить **краткую** по каждому модулю
- Перевести разработку на язык программирования go
- Новую разработку каждого модуля начать с чистого листа, опираясь на API и документацию.
